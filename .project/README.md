# Genie Autopilot

Genie Autopilot extends Genie Code with background AI agent functionality to take care of maintenance for jobs, pipelines, and other Databricks resources. Maintaining data pipelines is what data engineers spend over 50% of their time on, i.e. upgrades, migrations, security updates, behavior changes, data evolution, failures. Genie Autopilot addresses the toil so data and AI engineers can focus on making a difference!
